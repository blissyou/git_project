1. 깃은 저장,관리할 수 있고 깃허브는 깃을 호스팅해준다
2. Cd - change directory -> 디렉토리 변경해준다
3. Git init - .git 폴더를 만들어 로컬 레포지토리로 만들어준다
4. Remote add - 원격 저장소를 로컬 저장소와 연결해주는 명령어
5. Git add Quiz.txt
Git add로 변경하상이 있는 파일을 추가가 가능하지만 위 사진인 경우 Quiz.txt만 추가되어있기 때문
6. 원격 레포지토리에 변경사항이 있어서 지금 로컬 레포지토리의 변경사항을 업로드 못하는 경우
7. 따로따로 개발해 편리하다
8. Main -> git checkout ONEZERO
9. Git checkout , git merge ONEZERO , git push
